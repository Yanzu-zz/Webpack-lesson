## Webpack-lesson
  一个 webpack 配置练习项目

### 项目结构功能
```
  ├─Webpack_lesson
  │  ├─make-bundler 自己写一个 bundler
  │  ├─make-loader  自己写一个 loader
  │  ├─make-plugins 自己写一个 plugin
  │  ├─mutipage     多页打包配置
  │  ├─my-app       webpack 配置练习
  │  ├─my-project   配置主流框架（这里是Vue）打包配置
  │  └─type-script  打包 typescript 配置
```

** 具体内容请点开文件查看 **

## 安装与运行
```
  cd ./dirName/ && npm install
  npm run serve
```

## 打包
```
  npm run build
```
